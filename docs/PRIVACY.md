# Vernest Privacy Policy Draft

Vernest is designed as a local-first desktop voice input tool.

- Audio is captured and recognized locally.
- Recognition does not require cloud APIs.
- The app does not upload audio, transcripts, logs, crash reports, or diagnostics.
- Logs are stored locally under `%APPDATA%\Vernest\logs`.
- Diagnostic files are generated only when the user clicks export.
- Exported diagnostics remain local files until the user manually shares them.

This draft should be reviewed before any public commercial release.
