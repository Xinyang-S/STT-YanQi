# Vernest Privacy Policy Draft

Vernest is designed as a local-first desktop voice input tool, not an absolute
no-network product commitment.

- Audio is captured and recognized locally by default.
- The default recognition path does not require cloud APIs.
- Vernest does not automatically upload audio, transcripts, logs, crash reports,
  or diagnostics.
- Any future cloud enhancement must be explicitly enabled by the user and
  documented in this policy before release.
- Logs are stored locally under `%APPDATA%\Vernest\logs`.
- Diagnostic files are generated only when the user clicks export.
- Exported diagnostics remain local files until the user manually shares them.

This draft should be reviewed before any public commercial release.
